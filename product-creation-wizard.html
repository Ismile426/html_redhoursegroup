Walk through Catalog → Template (range #, brand, fixed spec, Ø/L + add-on pools) → Variants

Catalog Explorer Product Wizard

A Template is a range with its own number (e.g. 40400) — brand(s), country/company release, and the parts that never change across the range (Head, Drive, Point, Material, Coating) live here as single fixed values. Diameter & Length do vary within a range (confirmed from the RedHorse characteristics sheet — one range lists several Ø/L combos), so they're a pool on the template and picked per variant. Each Variant gets its own item number by appending a running suffix to the template — 40400 → 404001, 404002… If a variant also needs a Configuration/Colour/Size/Style add-on, that's appended with dashes: 404001-1-12-2-3.
